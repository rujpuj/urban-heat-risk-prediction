# Notebooks

Suggested notebook flow:

1. `01_eda.ipynb`: dataset summary, missing values, class distribution, and feature plots.
2. `02_feature_engineering.ipynb`: create or test new heat-risk indicators.
3. `03_model_experiments.ipynb`: compare Logistic Regression, Decision Tree, RandomForest, and XGBoost.

The main runnable project is implemented in `src/` and `app.py`.
